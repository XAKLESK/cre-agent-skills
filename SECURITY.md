# Security Policy

## About This Project

This repository contains Markdown skill files and knowledge base documents for commercial real estate analysis. There is no executable code, no API endpoints, no databases, and no user authentication.

## Reporting a Vulnerability

If you discover a security concern related to this project — for example, sensitive data accidentally committed, or a link that redirects to a malicious site — please report it by emailing:

**ahacker@theaiconsultingnetwork.com**

You will receive a response within 48 hours. If the issue is confirmed, a fix will be published and credited to the reporter (unless anonymity is requested).

## Scope

Since this project contains only Markdown documentation files, the attack surface is limited to:

- Malicious links embedded in skill files
- Sensitive data accidentally included in sample templates
- Phishing or impersonation concerns

If you find any of the above, please report it.
